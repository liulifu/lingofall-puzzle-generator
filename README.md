# lingofall-puzzle-generator

A source-available crossword puzzle generator resource published by Liulifu for noncommercial use.

This resource contains the puzzle generation engine used to build compact crossword-style puzzle data from word candidates. It is published as a standalone reference/resource package, separate from the lingofall website.
